# 问介AIer小程序后台管理系统

## 项目简介

这是一个专为「问介AIer」微信小程序设计的Web后台管理系统，基于微信云开发平台，支持案例、流程、资料等内容的管理，以及启动页、首页、导航栏、背景等配置项的编辑。

## 功能特性

### 内容管理
- 📁 **案例管理** - 案例列表增删改查，案例详情富文本编辑
- ⚙️ **流程管理** - 流程列表增删改查，流程详情步骤编辑
- 📚 **资料管理** - 资料列表增删改查，资料详情富文本编辑

### 配置管理
- 🚀 **启动页配置** - LOGO文字、副标题、品牌文字、背景视频
- 🏠 **首页配置** - 品牌名称、服务项目编辑
- 📱 **底部导航栏** - 导航项图标、文字、页面路径配置
- 🎨 **全局背景** - 图片/视频背景切换与上传
- 🔗 **分享配置** - 默认分享标题与封面设置

### 技术特点
- ☁️ 基于微信云开发Web SDK，数据实时同步
- 📤 支持图片、视频直接上传到云存储
- 📝 可视化富文本编辑器，拖拽排序内容块
- 📱 响应式设计，支持移动端访问
- 🔒 简单的密码保护机制

## 部署指南

### 方式一：GitHub Pages（推荐）

1. **创建GitHub仓库**
   - 登录 GitHub，点击右上角 "+" -> "New repository"
   - 仓库名称填 `wenjie-admin`
   - 选择 Public
   - 点击 "Create repository"

2. **上传文件**
   ```bash
   # 在本地admin目录执行
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/你的用户名/wenjie-admin.git
   git push -u origin main
   ```

3. **启用GitHub Pages**
   - 进入仓库 Settings -> Pages
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main"，文件夹选择 "/ (root)"
   - 点击 Save

4. **访问后台**
   - 等待1-2分钟，访问 `https://你的用户名.github.io/wenjie-admin/`

### 方式二：Netlify

1. 访问 [Netlify](https://www.netlify.com/)
2. 使用GitHub登录
3. 点击 "Add new site" -> "Import an existing project"
4. 连接GitHub仓库
5. 部署设置：
   - Build command: （留空）
   - Publish directory: `admin`
6. 点击 "Deploy site"
7. 访问生成的URL

### 方式三：Vercel

1. 访问 [Vercel](https://vercel.com/)
2. 使用GitHub登录
3. 点击 "Add New..." -> "Project"
4. 导入GitHub仓库
5. Framework Preset 选择 "Other"
6. 点击 "Deploy"

### 方式四：直接部署到服务器

将 `admin` 文件夹下的所有文件上传到你的Web服务器（如Nginx、Apache）即可。

## 使用说明

### 登录

1. 访问后台管理页面
2. 输入管理员密码登录
3. **默认密码**: `admin123`

> ⚠️ **重要**: 首次使用后，请务必修改默认密码！修改方式：在 `js/app.js` 文件中找到 `ADMIN_PASSWORD` 变量并修改。

### 数据初始化

首次登录后，系统会自动检查并初始化全局配置。如需初始化示例数据，可使用云开发控制台导入 `initial_data` 文件夹下的JSON文件。

### 内容管理

#### 添加案例
1. 进入「案例列表」页面
2. 点击「添加案例」按钮
3. 填写标题、描述、标签等信息
4. 上传封面图片
5. 点击保存

#### 编辑案例详情
1. 进入「案例详情」页面
2. 从下拉框选择要编辑的案例
3. 编辑基本信息（标题、标签、简介、引用等）
4. 使用内容块工具添加：
   - 📝 文本段落
   - 📌 二级标题 / 📍 三级标题
   - 🖼️ 单图 / 双图 / 三图
   - 💬 引用文字
   - ➖ 分隔线
   - ⬆️ 间距
5. 点击保存

### 配置管理

#### 修改全局背景
1. 进入「全局背景」页面
2. 选择背景类型（图片或视频）
3. 上传新的背景文件
4. 点击保存

> 💡 修改后，小程序所有页面会自动同步使用新背景。

#### 配置导航栏
1. 进入「底部导航栏」页面
2. 修改各导航项的图标、文字、路径
3. 点击保存

1. 进入 [云开发控制台](https://cloud.console.qq.com/)
2. 选择环境 `cloud1-d7g8ol36z34e7a782`
3. 进入「数据库」-> 「权限设置」
4. 推荐设置为：
   - 单记录查询/新增：`仅创建者及管理员可写，所有人可读`
   - 集合级别读写：`所有人可读，仅创建者及管理员可写`

## 安全建议

1. **修改默认密码** - 将 `js/app.js` 中的 `ADMIN_PASSWORD` 修改为强密码
2. **限制访问** - 可以通过Nginx配置Basic Auth增加一层保护
3. **定期备份** - 定期导出云开发数据库中的数据
4. **监控日志** - 关注云开发控制台的访问日志

## 常见问题

### Q: 上传图片失败怎么办？
A: 检查云开发的存储权限设置，确保已开通云存储服务。

### Q: 数据没有保存？
A: 查看浏览器控制台是否有错误信息，确认云开发环境配置正确。

### Q: 视频无法播放？
A: 微信云存储的视频需要转换为可访问的CDN链接，确保视频格式为MP4。

### Q: 如何添加新的内容块类型？
A: 修改 `js/app.js` 中的内容块相关函数，添加对应的渲染和保存逻辑。
